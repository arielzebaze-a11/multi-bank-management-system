# Système de Gestion Bancaire Multi-Banque

API bancaire multi-banque développée avec Node.js, Express et PostgreSQL.

## Fonctionnalités

- Authentification JWT
- Gestion des rôles
- Gestion des comptes
- Dépôts
- Retraits
- Virements
- Historique des transactions
- Génération de RIB PDF
- Rapports financiers PDF
- Administration complète

## Technologies

- Node.js
- Express.js
- PostgreSQL
- Sequelize
- JWT
- Swagger
- Jest
- Supertest
